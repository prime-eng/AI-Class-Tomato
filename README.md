# AI-Class-Tomato

Sistem klasifikasi tomat menggunakan YOLOv8.

## Fitur

### Kematangan

* ripe
* unripe

### Penyakit

* Anthrocnose
* Bacterial_Spot
* Botrytis
* fruitworm_bore
* tomato_cracking

## Menjalankan GUI

```bash
python gui.py
```

## Menjalankan API

```bash
python app.py
```
